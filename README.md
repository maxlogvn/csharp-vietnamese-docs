# C# Study - Tài liệu học C# tiếng Việt

Bộ tài liệu học ngôn ngữ C# dành cho người nói tiếng Việt, được dịch và biên soạn lại từ [tài liệu chính thức của Microsoft](https://learn.microsoft.com/dotnet/csharp/).

## Về dự án

Dự án này tập trung vào việc chuyển ngữ tài liệu C# từ tiếng Anh sang tiếng Việt với phong cách thân thiện, dễ hiểu, phù hợp cho lập trình viên mới bắt đầu. Nội dung được trình bày chi tiết, có ví dụ minh hoạ và hạn chế sử dụng thuật ngữ chuyên ngành phức tạp.

## Cấu trúc

```
csharp-study/
├── docs/
│   ├── en/          # Tài liệu gốc tiếng Anh (nguồn tham khảo)
│   └── vi/          # Bản dịch tiếng Việt
├── csharp-study/    # Dự án console app để thực hành
├── AGENTS.md        # Hướng dẫn dịch thuật
└── README.md
```

## Cách dùng

- Duyệt tài liệu trong thư mục `docs/vi/` để đọc các bài hướng dẫn bằng tiếng Việt.
- Chạy ví dụ minh hoạ bằng lệnh:

```bash
dotnet run --project csharp-study
```

## Đóng góp

Xem `AGENTS.md` để biết quy tắc dịch thuật nếu bạn muốn tham gia dịch tài liệu.
